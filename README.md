# SuiteQL Query Tool
v2025.1

A lightweight, zero-dependency Suitelet for running direct SQL queries within NetSuite. Built for developers who need raw data access without the overhead of Saved Searches.

### Quick Start
Download: Get the latest suiteql-tool-sl.js.
Upload: Save the file to the NetSuite File Cabinet (SuiteScripts/).
Deploy:
Create a Script Record (Customization > Scripting > Scripts > New).
Deploy it.
Open the generated URL.

### Features
Single File: No complex bundles or dependencies. Just one JS file.
Instant Execution: Type SQL, get results. No UI lag.
Modern UI: Clean, responsive interface using Tailwind CSS (CDN).
Zero Overhead: Bypasses the slow Saved Search UI entirely.
