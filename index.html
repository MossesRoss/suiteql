Deployment Guide

  1. 1

    Download

    Get the suiteql-querytool.js file from the Overview tab.

  2. 2

    Upload to File Cabinet

    Navigate to Documents > Files > SuiteScripts and upload.

  3. 3

    Create Script Record

    Go to Customization > Scripting > Scripts > New.

  4. ✓

    Deploy & Run

    Create the deployment and click the generated link.
